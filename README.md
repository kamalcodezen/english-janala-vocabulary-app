📘 Project Overview

English Janala একটি Fully Interactive Vocabulary Learning Web Application যেখানে ব্যবহারকারীরা Lesson অনুযায়ী শব্দ শিখতে পারে, শব্দের বিস্তারিত দেখতে পারে, উচ্চারণ শুনতে পারে এবং Dark Mode ব্যবহার করতে পারে।

এই প্রজেক্টে Live API ব্যবহার করে Dynamic UI তৈরি করা হয়েছে এবং পুরো অ্যাপ Async JavaScript দিয়ে পরিচালিত।
